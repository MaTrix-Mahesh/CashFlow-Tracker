# 💼 CashFlow Tracker

### Developed as a Project for **ProDesk IT**

---

## 📌 Project Overview

**CashFlow Tracker** is a web-based financial management application designed to help users efficiently track their income and expenses. The system provides real-time insights into spending patterns and remaining balance, enabling better financial planning and control.

---

## 🎯 Objective

The main objective of this project is to:

* Provide a simple and user-friendly interface for managing finances
* Track income and expenses in real-time
* Help users maintain financial discipline and budgeting

---

## 🚀 Key Features

### 🔹 Core Functionalities

* Add and manage **monthly salary**
* Record **multiple expenses**
* Automatic calculation of:

  * Total Expenses
  * Remaining Balance

---

### 🔹 Advanced Functionalities

* 💾 **Data Persistence using LocalStorage**

  * Data remains محفوظ even after page reload
* 🗑 **Delete Transactions**

  * Remove expenses instantly
* 🌙 **Light/Dark Mode Toggle**
* ⚠️ **Low Balance Alert System**

---

## 🧠 System Working

The system follows a simple financial formula:

Remaining Balance = Salary - Total Expenses

* User inputs salary
* Expenses are stored dynamically
* Total is calculated using JavaScript logic
* UI updates instantly without page reload

---

## 🛠️ Technology Stack
